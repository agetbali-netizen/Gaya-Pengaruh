Gaya Mempengaruhi

Saya menggunakan logika untuk meyakinkan orang lain
Saya biasanya menggunakan fakta-fakta untuk meyakinkan orang lain
Saya menggunakan kemampuan dan pengalaman saya untuk membujuk orang lain
Saya biasanya tetap teguh pada ide saya agar didengar walaupun orang lain tidak setuju
Saya men-challenge orang lain dengan menanyakan ide lain saat orang lain tidak setuju
Saya akan mendebat dan menekan orang lain agar mereka melihat dari sudut pandang Anda
Saya biasanya berkompromi dan memberikan kelonggaran untuk mendapatkan hasil yang saya inginkan
Saya biasanya melakukan pertukaran untuk mendapatkan hasil yang saya inginkan
Jika diperlukan, saya akan menunjuk diskusi sampai waktu yang memungkinkan
Saya menyakinkan orang lain dengan menceritakan kemungkinan positif apabila orang lain melakukan ide Anda
Saya menggunakan pendekatan inspiratif, cerita dan metafora untuk meyakinkan orang lain
Saya menggunakan posisi saya untuk mengkomunikasikan misi yang ingin saya capai
Saya mempengaruhi orang lain dengan menyatakan orang lain atau berhubungan baik dengan orang lain
Saya memiliki hubungan timbal balik yang menguntungkan bagi kedua belah pihak untuk meyakinkan orang lain
Saya menggunakan kedekatan personal dengan orang lain untuk mempengaruhi orang lain setuju dengan pandangan saya
